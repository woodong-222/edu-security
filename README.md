head에는 주소에 들어갈 이름과 css와 링크를 적었습니다.

body에는 메인 화면에 들어갈 각종 텍스트들을 넣었는데,
왼쪽 위부터 BCSD Security

배경화면, 팝업모달에 들어가는 색상도 넣었습니다.
