# Security_이성수

## 3주차과제

### / page

* 구성
1. 제일 위쪽에는 나만의 페이지 대표 이름(가칭)
2. 중간에는 명화와 나만의 명언
3. 제일 아래에는 HW의 인덱스

* 기능
로그인 페이지로 리다이렉트한다.

* 현재 인덱스 목록
  * 로그인 페이지
  * 과제2
  * 과제3

### / Login Page

* 구성
1. 제일 위쪽이 나만의 페이지 대표 제목(가칭)
2. 중간에 로그인이라고 크게 나오고, 로그인을 요청함.
아이디와 비밀번호 입력창과 로그인 버튼을 구현했다.
3. 마지막으로 자동 로그인 체크 박스와 
비밀번호 찾기를 할 수 있는 링크를 구현했다.

* 기능
  * 로그인 기능 구현
  * ID와 PW를 텍스트 박스에 입력
  * 로그인 기능은 4주차에 구현 예정

### 하고싶은 말
